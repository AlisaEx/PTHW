Command line introduction
from Learning Python the Hard Way
http://learnpythonthehardway.org/book/appendix-a-cli/ex1.html

LINUX/MAC OS
-------------------------------
pwd
	print working directory
cd
	change directory
ls
	list directory
mkdir
	make directory
pushd
	push directory
rmdir
	remove directory
popd
	pop directory
cp
	copy a file or directory
mv
	move a file or directory
less
	page through a file
cat
	print the whole file
xargs
	execute arguments
find
	find files
grep
	find things inside files
man
	read manual pages
apropos
	find what language is appropriate
env
	look at your environment
echo
	print some arguments
export
	export/set up a new environment variable
exit
	exit the shell
sudo
	become super root user
